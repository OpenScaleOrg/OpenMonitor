================================================================================
    MICROSOFT FOUNDATION CLASS LIBRARY: OpenMonitor Project Overview
================================================================================

This OpenMonitor application was created with the Application Wizard. It
demonstrates basic usage of the Microsoft Foundation Classes and serves as a
starting point for building Windows desktop applications.

This file provides a brief overview of each file that makes up the application.

OpenMonitor.vcxproj
    The main Visual C++ project file generated by the Application Wizard.
    Contains the Visual C++ version and information about the platforms,
    configurations, and project features selected.

OpenMonitor.vcxproj.filters
    The VC++ project filter file. Contains associations between project files
    and filters so that files with similar extensions are grouped under the
    same node in the Solution Explorer.

OpenMonitor.h
    The main application header file. Includes other project-specific headers
    (including Resource.h) and declares the COpenMonitorApp application class.

OpenMonitor.cpp
    The main application source file containing the COpenMonitorApp class.

OpenMonitor.rc
    A listing of all Microsoft Windows resources used by the program. Includes
    icons, bitmaps, and cursors stored in the RES subdirectory. This file can
    be edited directly in Microsoft Visual C++.

res\OpenMonitor.ico
    The icon file used as the application icon. Included in the main resource
    file OpenMonitor.rc.

res\OpenMonitor.rc2
    Contains resources that are not edited directly in Microsoft Visual C++.
    Any resources that cannot be edited by the Resource Editor should be
    placed in this file.

/////////////////////////////////////////////////////////////////////////////

The Application Wizard creates one dialog class:

OpenMonitorDlg.h, OpenMonitorDlg.cpp - Main dialog
    These files contain the COpenMonitorDlg class, which defines the
    behavior of the application's main dialog. The dialog template is in
    OpenMonitor.rc and can be edited in Microsoft Visual C++.

/////////////////////////////////////////////////////////////////////////////

Other features:

ActiveX Controls
    The application includes support for using ActiveX controls.

/////////////////////////////////////////////////////////////////////////////

Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) named
    OpenMonitor.pch and a precompiled types file named StdAfx.obj.

Resource.h
    Standard header for defining new resource IDs. Read and updated by
    Microsoft Visual C++.
