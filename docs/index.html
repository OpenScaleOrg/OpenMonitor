What it monitors

Everything. At once.
Nothing you don't need.

Six metrics, one glance. No dashboards. No bloat. No subscriptions.

⚡

Taskbar Integration

Embeds directly into the Windows taskbar. Stats stay visible regardless of which app has focus — no alt-tab, no overlay.

🌐

Network Speed

Live upload and download in bytes or bits per second. Auto-selects the active adapter, or pin to one manually.

🔥

Hardware Temperatures

CPU, GPU, and motherboard temperatures via LibreHardwareMonitor. Know when your machine is running hot before it throttles.

🎨

Custom Skins

Pick from built-in themes or author your own with a plain XML file. Colors, fonts, layout, item order — all yours.

📊

Traffic History

Logs daily, weekly, and monthly network usage locally on your machine. No account, no cloud, no expiry.

🔌

Plugin System

Extend the display with custom DLL plugins. Drop one into the plugins folder and it loads on next start. SDK included.
